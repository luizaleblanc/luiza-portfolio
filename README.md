## Nova Atualização: Arquitetura Bilíngue Dinâmica
O portfólio agora conta com um sistema de internacionalização (i18n) inteligente construído do zero. 
A aplicação detecta automaticamente o idioma padrão configurado no computador/navegador do usuário (`navigator.language`) e renderiza instantaneamente a interface em **Português** ou **Inglês**. 
* **Como funciona:** Integração fluida entre o roteamento estático do Astro (`.astro`) e o gerenciamento de estado do React (`.tsx` com `client:load`), permitindo tradução em tempo real sem recarregamento de página, preservando 100% o design e a performance.
